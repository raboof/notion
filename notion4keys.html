META key

All keybindings are activated by pressing the META key, which by default is the 'windows' key.
The 'grey' bindings are activated by pressing ALTMETA (by default pressing Shift while holding META).

Screen
Mod4+EscapeGo to next screen on multihead setup.
Shift+Mod4+EscapeGo to previous screen on multihead setup.
Mod4+1Switch to object 0 (workspace, full screen client window) within current screen.
Shift+Mod4+1Go to screen 0 on multihead setup.
Mod4+2Switch to object 1 (workspace, full screen client window) within current screen.
Shift+Mod4+2Go to screen 1 on multihead setup.
Mod4+3Switch to object 2 (workspace, full screen client window) within current screen.
Shift+Mod4+3Go to screen 2 on multihead setup.
Mod4+4Switch to object 3 (workspace, full screen client window) within current screen.
Shift+Mod4+4Go to screen 3 on multihead setup.
Mod4+5Switch to object 4 (workspace, full screen client window) within current screen.
Mod4+6Switch to object 5 (workspace, full screen client window) within current screen.
Mod4+7Switch to object 6 (workspace, full screen client window) within current screen.
Mod4+8Switch to object 7 (workspace, full screen client window) within current screen.
Mod4+9Switch to object 8 (workspace, full screen client window) within current screen.
Mod4+0Switch to object 9 (workspace, full screen client window) within current screen.
Mod4+TabForward-circulate focus.
Shift+Mod4+TabBackward-circulate focus.
Shift+Mod4+RRaise focused object, if possible.
Mod4+DToggle floating dock.
Mod4+graveSwitch to next object (workspace, full screen client window) within current screen.
Shift+Mod4+asciitildeSwitch to previous object (workspace, full screen client window) within current screen.
Mod4+spaceToggle scratchpad.
Mod4+F1Go to screen 0 on multihead setup.
Mod4+F2Go to screen 1 on multihead setup.
Mod4+F3Go to screen 2 on multihead setup.
Mod4+F9Create a new workspace of chosen default type.
F12Display the main menu.
Mod4+F12Display the main menu.
Client windows
Shift+Mod4+QSend next key press to the client window. Some programs may not allow this by default.
Shift+Mod4+LNudge the client window. This might help with some programs' resizing problems.
Shift+Mod4+CKill client owning the client window.
Mod4+QToggle client window group full-screen mode
Frames
Mod4+RBegin move/resize mode.
Shift+Mod4+HMaximize the frame horizontally.
Shift+Mod4+VMaximize the frame vertically.
Mod4+Caps_LockSwitch to next object within the frame.
Shift+Mod4+Caps_LockSwitch to previous object within the frame.
Mod4+ASwitch to tab 0 in this frame.
Mod4+SSwitch to tab 1 in this frame.
Mod4+DSwitch to tab 2 in this frame.
Mod4+FSwitch to tab 3 in this frame.
Shift+Mod4+HMaximize the frame horizontally/vertically.
Shift+Mod4+VMaximize the frame horizontally/vertically.
Mod4+NAttach tagged objects to this frame.
Shift+Mod4+commaMove current tab to the right within the frame.
Shift+Mod4+periodMove current tab to the left within the frame.
Tilings
Mod4+WGo to frame below current frame.
Shift+Mod4+WGo to frame above current frame.
Mod4+ISplit current frame horizontally.
Shift+Mod4+ISplit current frame vertically.
Mod4+XDestroy current frame.
Tabs

Shift+Mod4+TClear all tags.

Mod4+JQuery for command line to execute.
Shift+Mod4+JQuery for Lua code to execute.
Mod4+LLock screen
Mod4+CDisplay context menu.

Shift+Mod4+F4Query for host to connect to with SSH.
Shift+Mod4+F5Query for file to edit.
Shift+Mod4+F6Query for file to view.
